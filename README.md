# CARE-E Admin Panel

A simple, minimalistic admin page to view and review organization registration form submissions.

## Files

- `admin.html` - Main admin page
- `admin.js` - JavaScript for displaying form data

## Features

- View all organization submissions in a table
- Each submission has a unique ID (ORG001, ORG002, etc.)
- Click "View Details" button to see full information
- Shows total count of submissions
- Simple, beginner-friendly design

## How to Use

1. Open `admin.html` in any web browser
2. You'll see a table with all organization submissions
3. Click "View Details" to expand and see full information for any organization

## Sample Data

The page currently shows 5 sample organizations:
- City General Hospital
- LifeCare Clinic
- Central Blood Bank
- MediLab Diagnostics
- Apollo Regional Hospital

## Future Enhancements

To connect to real form submissions, you would need to:
1. Set up a backend server (Node.js, Python, etc.)
2. Create a database to store form submissions
3. Update the JavaScript to fetch real data from your server
